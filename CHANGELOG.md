# Changelog

## v3.0.0 - 2026-04-26

Public release of `openclaw-skill-guard`.

### Highlights

- Chinese-first Windows GUI as the main product entry point
- CLI retained for automation and advanced users
- OpenClaw-aware skill scanning with structured context, install-chain analysis, invocation policy analysis, reachability, prompt/instruction analysis, attack-path reasoning, consequence modeling, guarded runtime validation, suppression/audit, corpus-backed analyzers, dependency audit, URL/API classification, source/domain reputation hints, config/control-plane audit, capability manifest, companion-doc audit, and source-identity mismatch signals
- canonical JSON report with SARIF, Markdown, and HTML derived exports
- public repository cleanup to remove reference repositories, research snapshots, old demo reports, and internal planning documents
- Windows GUI release zip prepared for direct download

### Safety boundary

- not an exploit runner
- not an online reputation platform
- not a dynamic malware sandbox
- guarded validation stays non-executing
